```
>=python 3.7
pip3 install -r requirements.txt
```

# Usage
在netease.py同级目录下的music_list.txt文件中填入要下载的歌曲名（每行一个，参考项目中的示例），然后执行命令：
```
$ python netease.py
```

音乐会下载到同级目录的Musics文件夹中，Enjoy it！
